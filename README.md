# 西行漫记 · 新疆北疆自驾亲子游看板

这是一个为 **10人新疆自驾游 (教师家属亲子团)** 精心制作的交互式旅行规划看板。

## 🌟 看板亮点
- **极速国内访问**：采用系统内置高质量字体与纯原生 HTML5/CSS3/ES6 交互逻辑，**零外部强依赖阻塞**，国内网络环境秒开，完美解决加载缓慢的问题。
- **降本自驾方案**：内置以 **1辆国产SUV + 1辆大众捷达** 为核心的降本方案，对比天价包车与大商务车，每天可省去数百元租金与巨大油耗。
- **三套玩法随心切换**：
  - ① **轻松自驾版**：打卡经典北疆精选环线，节奏松紧适宜。
  - ② **极度自由休闲版**：舍去喀纳斯超长车程，深玩赛里木湖发呆2天，住伊宁六星街和琼库什台童话木屋。
  - ③ **亲子慢游探险版**：针对低幼儿童设计的极低车程版，霍城DIY薰衣草皂、那拉提骑马喂羊、抓土拨鼠、尝手工冰淇淋。
- **动态算账器**：内置车费与房费人均估算器，输入租金与民宿单价，自动计算分摊金额，方便老师记账。
- **行李与抢票备忘**：专为捷达/SUV搭配定制了“软行李袋打包装车”策略以及那拉提等核心景区“提前7天0点实名抢自驾票”的时间节点。勾选清单状态会自动本地持久化保存，关闭网页也不会丢。

## 🚀 后续如何一键发布到 GitHub Pages（国内直接查看）
当你满意本地效果后，可以通过以下简单命令将代码发布到 GitHub，从而获得一个能在手机上直接扫码或点击查看的专属网址：

### 第一步：初始化 Git 并提交
在项目根目录打开 PowerShell 并运行：
```powershell
git init
git add .
git commit -m "feat: init xinjiang travel dashboard"
```

### 第二步：推送到你的 GitHub 仓库
在 GitHub 上创建一个新仓库（例如 `xinjiang-tour`），然后运行：
```powershell
git remote add origin https://github.com/你的用户名/xinjiang-tour.git
git branch -M main
git push -u origin main
```

### 第三步：开启 GitHub Pages
1. 打开你的 GitHub 仓库页面，点击 **Settings** (设置) -> **Pages**。
2. 在 **Build and deployment** 下的 **Source** 选择 `Deploy from a branch`。
3. 在 **Branch** 选择 `main` 分支，点击右侧的 **Save** (保存)。
4. 稍等 1-2 分钟，刷新页面，GitHub 就会提供一个类似于 `https://你的用户名.github.io/xinjiang-tour/` 的国内可直接秒开的在线网址，发到微信群里大家就能在手机上完美查看了！
